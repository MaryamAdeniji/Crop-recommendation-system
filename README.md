# Crop-recommendation-system
This project uses supervised and unsupervised Machine learning techniques to provide recommendations of crops using soil characteristics such as Nitrogen, Phosphorous, Potassium, pH value and weather conditions such as rainfall and humidity. The aim of this project is to show how data-driven methods can support agricultural food production.
The proposed methodology and plan of action for this study follows the industry standard practice of data mining (CRISP-DM). They are listed below as follows:
•	Business Understanding.
•	Data Understanding and Collection.
•	Data Cleansing and Preparation.
•	Model Selection and Training.
•	Model Evaluation.
•	Deployment
The supervised machine learning approach is the classification techniques with the use of such as Support Vector Machine (SVM), Multilayer Perceptron (MLP), Deep Learning Convolution Neural Network (CNN). The unsupervised learning clustering techniques was also applied in the development of these prediction system. Having developed and tested each model, the prediction accuracy is evaluated and compared to determine which is more effective. The metrics to be used for comparison are accuracy, precision, recall, F1-score.
